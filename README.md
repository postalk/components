# postalk components

## Project setup

```
yarn
```

### Start storybook and hot-reloads for development

```
yarn start
```

### Lints and fixes files

```
yarn lint
```

### Run your unit tests

```
yarn test
```

If you want to update snapshots

```
yarn test -u
```
